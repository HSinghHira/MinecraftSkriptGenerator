## 📖 What is this?

This advanced Skript generator helps Minecraft server owners create professional, feature-rich custom commands. Convert your Minecraft commands into fully-featured Skript files with permissions, cooldowns, aliases, sound effects, and much more - all through an intuitive web interface.

## 🚀 How to Use

### **Basic Usage**
1. Visit the [Skript Generator](https://git.hsinghhira.me/MinecraftSkriptGenerator/)
2. Enter your command name (e.g., "warrior", "kit", "spawn")
3. (Optional) Add command aliases separated by commas
4. Paste your Minecraft commands in the input area
5. Click **"🔨 Generate"** to create your Skript
6. Click **"💾 Save .sk"** to download the file

### **Advanced Configuration**
Navigate through the tabs to customize:
- **Basic** - Permissions, cooldowns, execution mode
- **Execution** - Delays, loops, coordinate replacement
- **Messages** - Success messages, sounds, display types
- **Advanced** - Code formatting, comments, auto-preview

### **Installation**
1. Download your generated `.sk` file
2. Upload to `plugins/Skript/scripts/` on your server
3. Run `/sk reload <filename>` or restart your server
4. Grant permission `<commandname>.give` to users
5. Use your command in-game: `/yourcommand`

## 📝 Example Outputs

### **Simple Command**
```skript
command /warrior:
    permission: warrior.give
    permission message: &cYou don't have permission!
    trigger:
        # Command 1
        make console execute command "give @p diamond_sword"
        send "&aAll items spawned!" to player
```

### **Advanced Command with Features**
```skript
# Generated by Minecraft Skript Generator
# Command: /kit
# Date: 10/19/2025

command /kit <alias:/k, /kits>:
    permission: kit.give
    permission message: &cYou don't have permission to use kits!
    cooldown: 300 seconds
    cooldown message: &cPlease wait %remaining time% before using this again!
    trigger:
        # Execute command 1

        wait 0.5 seconds
        # Execute command 2
        make console execute command "give @p diamond_armor{...}"
        # Send success notification
        send action bar "&aKit received successfully!" to player
        # Play success sound effect
        play sound "entity.player.levelup" to player
```

## 🐛 Troubleshooting

### **Command doesn't work in-game:**
- Verify Skript plugin is installed and enabled
- Check console for syntax errors: `/sk reload <filename>`
- Ensure you have the required permission
- Test with OP status first

### **Permission issues:**
- Grant permission: `<commandname>.give`
- Or use wildcard: `<commandname>.*`
- Check your permissions plugin configuration

### **Cooldown not working:**
- Ensure cooldown value is greater than 0
- Verify Skript version supports cooldowns
- Check if player has cooldown bypass permission

### **Commands execute incorrectly:**
- Verify original commands work in-game
- Check for quote conflicts (use single quotes inside commands)
- Test coordinate replacement if using relative positions
